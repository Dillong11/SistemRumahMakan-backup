SistemRumahMakan
=================

Backup of local `SistemRumahMakan` ASP.NET project.

Quick start
-----------

- Restore dependencies:

```
dotnet restore
```

- Build:

```
dotnet build --configuration Release
```

- Run (from project folder):

```
dotnet run
```

CI
--

This repository includes a GitHub Actions workflow that restores and builds the project on push and pull request.
